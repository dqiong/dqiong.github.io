发表于 2016-02-29   |   分类于 LeetCode   |  

Implement atoi to convert a string to an integer.

注意考虑以下情况:
“123” : 123
“-123” : -123
“00123/2” : 123
“-123/2”; : -123
“- 123/2” :0
“-+123/2” :0
“10522545459” : 2147483647 越界
“2147483647”; : 2147483647 刚好不越界
“-21474836480”; : -2147483648;越界
“-2147483648”; : -2147483648 刚好不越界
